1. i
This is used to edit stuff inside the file.

2. escape key
This will bring you back to normal mode.

3. :w
This is used to save the file. (w refers to write)

4. :q
This is used to quit vim.

5. dd
This is used to delete/cut a single line. (works in normal mode)

6. yy
Copy a line.

7. p
Paste a line.

8. u
undo something

9. :set number
This enables line numbers on the left side of the editor.

10. :!python3 example.py
Runs the Python file from inside Vim.

You can also use:
:!python3 %

% represents the current file name.
